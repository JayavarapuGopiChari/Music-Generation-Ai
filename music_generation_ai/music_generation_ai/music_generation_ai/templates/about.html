About this project

An end-to-end deep learning pipeline that learns musical patterns from MIDI files and composes new music, paired with real AI-generated cover art.

Tech stack

  • music21 — parsing MIDI files into notes/chords
  • TensorFlow / Keras — LSTM sequence model
  • Flask + SQLite — web interface and login/signup
  • Pollinations.ai — free text-to-image API for cover art
  • NumPy — numerical processing

Pipeline

  1. preprocess.py extracts note/chord events from every MIDI file in data/midi_songs/.
  2. train.py encodes those events into fixed-length sequences and trains a 3-layer LSTM network.
  3. generate.py seeds the trained network with a random snippet and autoregressively predicts new notes, shaped by your chosen Style (instrument + creativity) and Emotion (tempo + octave).
  4. Predicted notes/chords are converted back into a .mid file you can play or edit.
  5. image_gen.py sends your description + style/emotion to a free public text-to-image API to create matching cover art.

Note: the melody model responds to Style/Emotion tags (not the literal words in your description) — true text-understanding music generation needs much larger pretrained models than this project trains from scratch. The cover art, however, is genuine text-to-image generation from your full description.
